Security definitions

An identity provider (IdP) is a system that creates, stores and manages digital identities. Simply stated, an IdP is user authentication as-a-service.

Security Assertion Markup Language: a protocol for exchanging authentication and authorization information between an identity provider and a third party application.

The answer to the question: who are you?

The answer to the question: what are you allowed to do? Even possible without authentication, because in some cases an anonymous user can also access certain data and/or services.

A string that contains encrypted authentication data. As long as a token is valid, users will have access to a website or app for which the token has been created.

Open Authorization: an open standard for authorization. Users can use this to grant an application access to their data, which is stored in another website, without exposing the username and password.

A protocol to exchange authentication and authorization data between an identity provider and a third party application.

Multi Factor Authentication: an authentication method that forces an online user to complete two steps to gain access to something.

Integrated Windows Authentication: geïntegreerde Windows-verificatie maakt gebruik van de beveiligingsfuncties van Windows-clients en -servers. In tegenstelling tot basisauthenticatie of samenvattingsauthenticatie, wordt gebruikers in eerste instantie niet om een gebruikersnaam en wachtwoord gevraagd. Integrated Windows Authentication: integrated Windows verification uses Windows client and server security functionality. Unlike basic or ... authentication, users are not asked for a password at first hand.

Active Directory: and authentication and authorization service provided by Microsoft. In this context mainly used as a identity provider.

Single Signon: authenticating without again having to enter a username and password.

World Wildlife Foundation: a global organization dedicated to protecting nature. The giant panda is the organization symbol.

Rickrolling or a Rickroll: an internet meme involving the unexpected appearance of the music video for the 1987 song "Never Gonna Give You Up". The meme is a type of bait and switch, usually using a disguised hyperlink that leads to the music video. When victims click on a seemingly unrelated link, the site with the music video loads instead of what was expected, and they have been "Rickrolled".
